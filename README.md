## Что мы сделали

Полноценное веб-приложение для игры в шахматы с минималистичным дизайном и несколькими умными фичами под капотом.

**Игра**
- Партия против ИИ на базе движка Stockfish — 4 уровня сложности (от новичка до мастера)
- Игра вдвоём на одном устройстве — передай телефон другу и играйте

**ИИ-тренер**
- Встроенный чат-тренер на Llama 3 (8B) через Groq API: анализирует позицию на доске и даёт советы прямо во время партии — не после, а в момент

**Обучение**
- Интерактивные туториалы по дебютам, тактике и эндшпилю

**Аккаунт**
- Авторизация через Telegram — никаких форм и паролей, один клик

**Техническая часть**
- Next.js 14 + TypeScript + Tailwind CSS
- Supabase для хранения данных
- SEO-оптимизация: Open Graph, Twitter Card, мета-теги на русском и английском, индексируется поисковиками
- Адаптивный дизайн, тёмная тема
- Деплой на Vercel
